# Networking

Network configuration, file sharing, and Active Directory integration.

*Documentation coming soon.*

---

## Features

### Network Discovery
Enable/disable Avahi for automatic network device discovery.

### Samba File Sharing
Share folders with Windows machines and other devices on your network.

### Active Directory
Join your Linux machine to a Windows domain.

### Firewall
Basic firewall configuration.
