# Setup Tools

Setup Tools handles first-time configuration after installing Linux: multimedia codecs, graphics drivers, and essential applications.

*Documentation coming soon.*

---

## Quick Overview

### Multimedia Codecs
Install codecs for playing videos, music, and streaming content. Covers:
- H.264/H.265 video
- MP3, AAC, FLAC audio
- DVD playback

### Graphics Drivers
- NVIDIA proprietary drivers
- AMD drivers
- Intel graphics

### Essential Apps
Common utilities and applications for a complete desktop experience.
