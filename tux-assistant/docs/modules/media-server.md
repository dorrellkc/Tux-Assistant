# Media Server

Set up Plex, Jellyfin, or Emby to stream your media collection.

*Documentation coming soon.*

---

## Supported Servers

### Plex
Commercial media server with free tier. Best apps on smart TVs and devices.

### Jellyfin
Fully open source. No accounts, no tracking, no fees.

### Emby
Middle ground between Plex and Jellyfin.
