# Tux Tunes

Internet radio player with smart song recording.

*Documentation coming soon.*

---

## Features

### Internet Radio
Browse and play internet radio stations.

### Smart Recording
Automatically record songs as separate files.

### Library
Organize your recorded music.

---

## Launch

Tux Tunes is a separate app. Launch from:
- Application menu → Tux Tunes
- Terminal: `tux-tunes`
- Inside Tux Assistant: Media and Entertainment → Tux Tunes
