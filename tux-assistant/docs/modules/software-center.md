# Software Center

Browse and install applications by category.

*Documentation coming soon.*

---

## Categories

- **Browsers**: Firefox, Chrome, Brave, etc.
- **Development**: VS Code, Sublime Text, IDEs
- **Graphics**: GIMP, Inkscape, Blender
- **Office**: LibreOffice, OnlyOffice
- **Media**: VLC, MPV, Spotify
- **Gaming**: Steam, Lutris, Wine
- **Utilities**: Various system tools

---

## How It Works

1. Click a category to see available apps
2. Click an app to see details
3. Click Install to install

The app uses your system's package manager (pacman, apt, dnf, zypper) behind the scenes.
